# 📊 Análisis de Datos - Alura Store LATAM

## 📌 Descripción
Proyecto de análisis exploratorio de datos utilizando Python, Pandas y Matplotlib.

## 📊 Análisis realizados
- Ingreso total por tienda
- Ventas por categoría
- Valoración promedio
- Productos más y menos vendidos
- Costo de envío promedio
- Análisis geográfico (opcional)

## 🛠 Tecnologías
- Python
- Pandas
- Matplotlib
- Google Colab

## ▶️ Ejecución
1. Ejecutar las celdas en orden.
2. Analizar los resultados generados.
3. Leer el informe final.
